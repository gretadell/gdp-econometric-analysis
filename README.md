# GDP Econometric Analysis

This repository contains a university econometrics project focused on the analysis of GDP per capita using World Bank indicators.

The project applies OLS regression models to study the relationship between GDP per capita, gross savings, population growth, R&D expenditure, education expenditure, and rule of law.

The analysis includes data reshaping, log transformations, model comparison, residual diagnostics, and leverage-based filtering.

Several regression specifications are tested and compared using adjusted R-squared and OLS assumption checks.

Particular attention is given to residual normality, linearity, homoscedasticity, and influential observations.

The final models suggest a positive association between GDP per capita, R&D expenditure, and institutional quality.
